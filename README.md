## 📱 Project Overview

AutoCareAI is a smart mobile application designed to help users with vehicle management and care.  
It provides a structured and scalable foundation for future AI-based automotive assistance features.
The app is built using modern mobile development practices including component-based architecture, file-based routing, and cloud build deployment.

---

## 🚀 Features

- Cross-platform mobile application (Android & iOS)
- Clean and responsive UI
- Splash screen with branding
- File-based navigation using Expo Router
- Modular and reusable components
- Scalable project structure
- Ready for AI integration (future enhancement)

---

## 🛠️ Tech Stack

- React Native (Expo SDK 54)
- Expo Router
- JavaScript / TypeScript
- Node.js & npm
- EAS Build (Expo Application Services)
- Git & GitHub

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AutoCareAI.git
cd AutoCareAI

2. Install dependencies

npm install

3. Start development server

npx expo start

⸻

🔐 Environment Variables

Create a .env file in the root directory:

GEMINI_API_KEY=your_api_key_here

👉 DO NOT commit real API keys to GitHub.

Use .env.example instead:

GEMINI_API_KEY=your_key_here

⸻

📦 Build (Android / iOS)

This project uses EAS Build:

eas build -p android
eas build -p ios
```
